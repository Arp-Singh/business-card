## Overview

This is a simple business card design in pure HTML and CSS. It is a demo project exercises used to gain confidence in my CSS skills. It is made for web browsers on desktop **ONLY.**

### Features

Users should be able to:

- Click or press on the card itself and it flips around to the back.
- Hover over the card to see some animations.
- Have a short hand way of gaining information on me.
